lambda 表达式的语法格式如下：
  (parameters) -> expression 或 (parameters) ->{ statements; }

位运算符：
  左移( << ) 0000 0101 << 2 -> 0001 0100
  右移( >> ) 0000 0101 >> 2 -> 0000 0001;  1111 1011 >> 3 -> 1111 1111
  无符号右移( >>> )  1111 1011 >>> 3 -> 0001 1111
  位与( & )  0000 0101 & 0000 0011 -> 0000 0001
  位或( | )  0000 0101 | 0000 0011 -> 0000 0111
  位异或( ^ ) 0000 0101 ^ 0000 0011 -> 0000 0110
  位非( ~ ) ~ 0000 0101  ->  1111 1010
